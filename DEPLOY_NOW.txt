                    DEPLOY YOUR ERP TO PRODUCTION NOW
================================================================================

FASTEST OPTION: Vercel + Render (5 minutes)

================================================================================
STEP 1: DEPLOY FRONTEND TO VERCEL (2 minutes)
================================================================================

1. Go to: https://vercel.com/new
2. Click "Continue with GitHub"
3. Select your repository
4. Click "Import"
5. In "Environment Variables" section, add:
   - Name: VITE_API_URL
   - Value: https://your-backend.onrender.com (we'll get this in step 2)
6. Click "Deploy"
7. Wait for deployment to complete
8. Copy your frontend URL: https://your-app.vercel.app

================================================================================
STEP 2: DEPLOY BACKEND TO RENDER (2 minutes)
================================================================================

1. Go to: https://render.com/dashboard
2. Click "New +"
3. Select "Web Service"
4. Click "Connect account" and authorize GitHub
5. Select your repository
6. Fill in the form:
   - Name: production-management-backend
   - Environment: Node
   - Build Command: cd backend && npm install
   - Start Command: cd backend && npm run start:prisma
   - Root Directory: ./

7. Click "Advanced" and add these Environment Variables:
   PORT=5001
   NODE_ENV=production
   JWT_SECRET=your-secret-key-12345
   DATABASE_URL=postgresql://user:password@host/dbname
   CORS_ORIGIN=https://your-app.vercel.app

8. Click "Create Web Service"
9. Wait for deployment
10. Copy your backend URL: https://your-backend.onrender.com

================================================================================
STEP 3: SETUP DATABASE (1 minute)
================================================================================

Option A: Use Render PostgreSQL
1. In Render dashboard, click "New +"
2. Select "PostgreSQL"
3. Fill in:
   - Name: production-management-db
   - Database: erp_db
   - User: erp_user
4. Click "Create Database"
5. Copy the connection string
6. Use as DATABASE_URL in backend environment variables

Option B: Use Neon (Free)
1. Go to: https://neon.tech
2. Sign up
3. Create project
4. Copy connection string
5. Use as DATABASE_URL in backend

================================================================================
STEP 4: UPDATE FRONTEND WITH BACKEND URL (1 minute)
================================================================================

1. Go back to Vercel dashboard
2. Select your project
3. Go to Settings → Environment Variables
4. Update VITE_API_URL to your Render backend URL
5. Click "Save"
6. Redeploy frontend (Vercel will auto-redeploy)

================================================================================
STEP 5: TEST YOUR DEPLOYMENT (5 minutes)
================================================================================

1. Go to: https://your-app.vercel.app
2. Login with:
   Email: admin@example.com
   Password: password

3. Test these modules:
   - Go to Sales → Add Sale
   - Go to Procurement → Add Purchase
   - Go to Products → Add Product
   - Go to Users → View Users

4. Verify data saves and persists after refresh
