## QUICK START (5 MINUTES)

### Step 1: Frontend to Vercel
```
1. Go to: https://vercel.com/new
2. Connect GitHub
3. Select repository
4. Click Deploy
5. Get URL: https://your-app.vercel.app
```

### Step 2: Backend to Render
```
1. Go to: https://render.com
2. Create Web Service
3. Connect GitHub
4. Set environment variables
5. Click Deploy
6. Get URL: https://your-backend.onrender.com
```

### Step 3: Database
```
1. Use Render PostgreSQL or Neon
2. Get connection string
3. Add to backend environment variables
4. Done!
```

### Step 4: Test
```
1. Go to frontend URL
2. Login: admin@example.com / password
3. Test modules
4. Verify data saves
```
