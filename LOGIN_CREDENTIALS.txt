Company Admin User Management:

- PUT    /api/company-admin/users/:id          (Update user)

Super Admin:
- GET    /api/super-admin/companies            (List companies)

- GET    /api/super-admin/admins               (List admins)

Authentication:
- POST   /api/auth/login                       (Login with email & password)
- GET    /api/auth/me                          (Get current user)

================================================================================
                         TROUBLESHOOTING
================================================================================

If you can't login:
1. Make sure both frontend and backend are running
2. Check that the email and password are correct
3. Refresh the page and try again
4. Clear browser cache

If you see "User limit reached":
1. You've reached the maximum users for your subscription plan
2. Delete some users or upgrade your plan

If you see "Access Denied":
1. Make sure you're logged in as an admin
2. Check that you have permission to perform the action

If you see redirect loop:
1. Clear browser localStorage
2. Clear browser cache
3. Try logging in again
