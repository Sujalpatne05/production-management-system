## Navigation Tips

1. **Expand/Collapse** - Click on any module to expand/collapse sub-items
2. **Search** - Use Ctrl+F to search for specific modules
3. **Hover** - Hover over items to see descriptions
4. **Active State** - Current page is highlighted in blue
5. **Badges** - Some modules show item counts (e.g., Reports: 15+)

## Responsive Behavior

### Desktop (1920px+)
- Full sidebar visible
- All text and icons visible
- Descriptions shown on hover

### Tablet (768-1024px)
- Sidebar collapses to icons
- Hover to see module names
- Touch-friendly spacing

### Mobile (< 768px)
- Hamburger menu
- Full-screen navigation
- Optimized for touch
