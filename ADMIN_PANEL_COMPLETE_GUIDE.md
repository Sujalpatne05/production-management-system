## How to Navigate

### Via Sidebar
1. **Expand Department** - Click on department name
2. **View Sub-items** - Sub-items appear with animation
3. **Access Module** - Click on any sub-item
4. **Collapse** - Click again to collapse

### Via Modules Overview Page
1. **Navigate to Page** - Click "Modules" in sidebar
2. **View All Modules** - See all 23 modules in cards
3. **Click Module** - Navigate directly to module
4. **View Stats** - See module counts and statistics
